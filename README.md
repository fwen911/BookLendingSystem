# 图书借阅管理系统 - 前端实现

## 项目概述

这是一个简化版的图书借阅管理系统前端界面，采用纯HTML、CSS和JavaScript实现，无需后端服务即可运行。该界面提供了完整的用户体验，包括图书浏览、搜索、分类筛选、图书详情查看等功能。

## 主要功能

### 1. 导航系统
- 首页：展示所有图书列表
- 图书管理：预留功能入口
- 借阅记录：预留功能入口
- 个人中心：预留功能入口

### 2. 图书浏览
- 网格布局展示图书列表
- 每个图书卡片包含：封面、标题、作者、分类、借阅状态
- 支持点击借阅按钮提交借阅请求
- 支持点击详情按钮查看完整信息

### 3. 搜索与筛选
- 搜索框支持按图书名称、作者或ISBN搜索
- 分类标签支持按类别筛选图书
- 交互式反馈提示

### 4. 图书详情
- 模态框展示图书详细信息
- 包含：作者、ISBN、分类、出版日期、出版社、页数、状态、内容简介
- 支持关闭详情弹窗

### 5. 交互反馈
- 操作成功提示消息
- 导航切换状态反馈
- 分类筛选状态反馈
- 卡片悬停动画效果

## 技术特点

1. **纯前端实现**：无需后端服务，可直接在浏览器中运行
2. **响应式设计**：适配不同屏幕尺寸的设备
3. **现代UI设计**：采用简洁、美观的界面风格
4. **交互动效**：包含悬停效果、过渡动画等
5. **模拟数据**：内置6本经典图书的详细信息
6. **用户友好**：提供即时反馈和视觉引导

## 运行说明

### 方法一：直接打开HTML文件
1. 找到 `index.html` 文件
2. 双击打开或右键选择浏览器打开

### 方法二：使用Python HTTP服务器
1. 确保已安装Python
2. 打开命令提示符，导航到frontend目录
3. 运行命令：`python -m http.server 8081`
4. 打开浏览器访问：`http://localhost:8081`
